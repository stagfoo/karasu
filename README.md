# nim-playing

create an app using nim
- use nim as much as possible 
- htmx for reactive programing (https://htmx.org/)
- webview 

build webapp
```
nim c -o:bin/app src/app.nim
```

build webview wrapper
```
nim c -o:bin/viewer src/viewer.nim
```
